# Chandrakanth K — Portfolio Website

A modern, fully-responsive professional portfolio website built with **HTML, CSS & JavaScript**.

## 🗂️ Folder Structure

```
portfolio/
├── index.html              ← Home page
├── css/
│   └── style.css           ← Global stylesheet (variables, all components)
├── js/
│   └── main.js             ← All JS: loader, navbar, AOS, skill bars, form
├── pages/
│   ├── about.html          ← About page
│   ├── projects.html       ← Projects page
│   └── contact.html        ← Contact page
└── assets/
    ├── images/             ← Add profile photo here as "profile.jpg"
    └── icons/              ← Custom icons (if any)
```

## 🚀 Deployment

### GitHub Pages
1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: `main` branch, `/ (root)` folder
3. Your site will be live at `https://yourusername.github.io/repo-name/`

### Vercel
1. Drag-and-drop the `portfolio/` folder at [vercel.com/new](https://vercel.com/new)
2. No build step needed — static HTML

### Netlify
1. Drag-and-drop at [app.netlify.com](https://app.netlify.com)

## ✨ Features
- 🌗 Dark / Light mode toggle (persistent via localStorage)
- 📱 Fully responsive — mobile, tablet, desktop
- 🎞️ Scroll animations (custom AOS)
- ⚡ Animated skill bars
- ✅ Contact form with validation + simulated success
- 🔢 Animated stat counters
- ⌨️ Typed text hero effect
- 🏷️ Project filter tabs
- 🔝 Sticky navbar with active state

## 🎨 Customisation
- Update contact details & social links in each HTML file
- Add your profile photo to `assets/images/profile.jpg` and update the About page
- Update the resume download link in `about.html`
- Push to GitHub and share!

## 🛠️ Tech Stack
HTML5 · CSS3 (custom properties) · Vanilla JavaScript · Font Awesome 6 · Google Fonts (Syne + DM Sans)
